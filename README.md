# myroom-vr — 江戸の町、通りに面した奥の間

Meta Questのブラウザから入れる、江戸の商人町のVR空間。WebXR + Three.js + Vite + TypeScript。

**▶ 公開URL: https://lalalatang.github.io/myroom-vr/** (Questブラウザで開いて「ENTER VR」)

- 拠点は表通りに面した町家の**奥の間**(文机・行灯・掛け軸・ラジオ)。障子を開けて土間を抜けると、両側に商家が軒を連ねる**表通り(60m)**
- 通りは物売りの声・ざわめき・下駄音(すべてプロシージャル合成)と、歩く町人たち(簡易NPC 20体)で賑わう
- 朝/昼/夕/夜の切替(提灯は夕夜に自動点灯、夜は木戸が閉まり通りが静まる)、晴れ/雨(雨の日は人が減る)
- 通りの東の突き当たりに神社の石段と鳥居、南の路地に井戸端と物干し。遠景は町並みと山並みが地平まで続く
- アセットはすべてCC0(Poly Haven / ambientCG)。一覧は [ASSETS.md](ASSETS.md)。v1(和モダンの一軒家)は gitタグ/履歴に残っています

## 起動(ローカル開発)

```bash
npm install
npm run fetch-assets   # CC0アセット(HDRI・テクスチャ)をダウンロード(初回のみ)
npm run dev            # https://localhost:5173 (自己署名HTTPS。WebXRに必須)
```

デスクトップブラウザでの操作(開発検証用フォールバック):

| 操作 | キー |
|---|---|
| 移動 | クリックで操作開始 → **WASD**(Shiftで速く) |
| 視点 | マウス |
| 仕掛けの操作 | 画面中央を対象に合わせて**左クリック** |
| 時間帯 | **1**=朝 **2**=昼 **3**=夕 **4**=夜 |
| 天候 | **R** で晴れ⇔雨 |
| 小銭を投げる | **C**(賽銭箱に当たると音が変わる) |

※デスクトップ移動は開発用の簡易実装のため壁をすり抜けられます。VRのテレポートは歩行可能面(通り・路地・室内・床几)に制限されます。

## 【帰宅後にやること】Quest実機での確認手順

1. **同じWi-Fi**にPCとQuestを接続し、PCで `npm run dev`。表示されるLAN URL(例: `https://192.168.x.x:5173`)をQuestのブラウザで開く(証明書警告は「詳細設定」→続行)。公開URL版なら警告なし
2. 「**ENTER VR**」で入室
3. チェックリスト:
   - [ ] **移動**: 左スティック前倒しで放物線照準→離すとテレポート。右スティック左右でスナップターン45°
   - [ ] **奥の間**: 行灯・ランプの点灯 / ラジオ再生 / 掛け軸の歌めくり / 障子の開閉
   - [ ] **通りの賑わい**: 障子を開けると通りの音が大きくなる。通りに出るとNPCが歩いており、近づくと会釈する
   - [ ] **通りの仕掛け**: 井戸の釣瓶(路地の奥)/ 蕎麦屋台の湯気 / 小銭投げ(**右コントローラーのグリップ**)/ 夕方に提灯が点く / 夜に木戸が閉まる / 時間帯が変わる時に遠くで寺の鐘
   - [ ] **浮遊感がない**: どの方向を見ても世界の切れ目が見えない
   - [ ] 体感72fps(カクつき・酔いがない)

## ラジオ音源の差し替え方法

`public/audio/` に **mp3(ogg/m4a/wavも可)を置くだけ**。ファイル名順に連続再生。
プレイリストはViteが自動生成(JSON手編集不要)。音源が無い場合はプロシージャルの「深夜ラジオ風」音が流れます。
※公開リポジトリなので、公開してよい音源のみ置くこと。

## 短歌(掛け軸)の差し替え方法

`public/texts/tanka.json` を編集(形式は同ファイル参照。`lines` 配列+`author`)。掛け軸クリックで次の一首。

```bash

node scripts/verify-desktop.mjs /tmp
# ロード・WASD移動・時間帯/天候切替・ドローコール/三角形数をヘッドレスChromeで検証
```

## アーキテクチャ

- `src/core/` — モジュール間契約(types.ts)・レイアウト規約(layout.ts)・状態(state.ts)・イベントバス(events.ts)・インタラクション登録(registry.ts)
- `src/world/` — machiya(町家)/ street(表通り・商家・路地・鳥居・木戸)/ skyline(遠景リング・山並み)/ materials / geo
- `src/systems/` — XRコントローラー・テレポート・デスクトップ操作・照明(昼夜+フォグ)・天候・空間オーディオ(賑わい)・NPC
- `src/interactions/` — 障子・行灯・ラジオ・掛け軸・風鈴・手首パネル・通りの仕掛け(井戸・屋台・木戸・小銭)

設計上の仮定・決定は [DECISIONS.md](DECISIONS.md)、v2の完了報告は [REPORT_V2.md](REPORT_V2.md) を参照。
